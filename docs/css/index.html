    mixins

    font-size

    @mixin font-size($size) { ... }

    Description

    Use to assign font-size using rem with px size fall-back.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $size

    font-size in pixels

    		Px— none

    Example

    p {
  @include font-size(14px)
}

    Output

    p {
  font-size: 14px; //Will be overridden if browser supports rem
  font-size: 0.8rem;
}

    Requires

    • [function] calculateRem

    icon

    @mixin icon($code) { ... }

    Description

    Use this to include all the necessary icon style attributes by only passing through the icon code. Note that the "\" should not be included in the unicode string.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $code

    the content code of the icon

    		String— none

    Example

    a.pdf:before {
  @include icon("e900");
}

    Output

    a.pdf:before {
  font-family: 'q4-icons';
  speak: none;
  font-style: normal;
  font-weight: normal;
  font-variant: normal;
  text-transform: none;
  line-height: 1;
  // Better Font Rendering
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
  content: "\e900";
}

    Requires

    • [function] unicode
    • [variable] icon-font

    layout-large

    @mixin layout-large() { ... }

    Description

    Used to assign styling for certain breakpoints quickly. Has 3 other types: layout-legacy / layout-medium / layout-small

    Parameters

    None.

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    .sidebar {
  width: 60%;
  float: left;
  margin: 0 2% 0 0;
  @include layout-small {
    width: 100%;
    float: none;
    margin: 0;
  }
}

    fit-and-center

    @mixin fit-and-center($width) { ... }

    Description

    used to set a max-width as well as center an element

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $width

    the value to be assigned to the max-width attribute

    		Px— none

    Example

    .pane_inner {
  @include fit-and-center($layout-width);
}

    Output

    .pane_inner {
  max-width: 940px;
  margin-left: auto;
  margin-right: auto;
}

    sr-only

    @mixin sr-only() { ... }

    Description

    used to quickly apply screen-reader-only styling

    Parameters

    None.

    Example

    .module_label {
  @include sr-only();
}

    Output

    .module_label {
  position: absolute;
  width: 1px;
  height: 1px;
  padding: 0;
  margin: -1px;
  overflow: hidden;
  clip: rect(0,0,0,0);
  border: 0;
}
