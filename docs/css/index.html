    @function calculateRem($size) { ... }

    Description

    A function used to convert rem font sizes to pixel size. This is a supporting function for the font-size mixin.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value

    font-size in pixels

    Used by

    mixins

    font-size

    @mixin font-size($size) { ... }

    Description

    Use to assign font-size using rem with px size fall-back.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $size

    font-size in pixels

    		Px—none

    Output

    p {
  font-size: 14px; //Will be overridden if browser supports rem
  font-size: 0.8rem;
}

    Example

    p {
  @include font-size(14px)
}

    Requires

    • [function] calculateRem

    layout-large

    @mixin layout-large() { ... }

    Description

    Used to assign styling for certain breakpoints quickly. Has 3 other types: layout-legacy / layout-medium / layout-small

    Parameters

    None.

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Example

    .sidebar {
  width: 60%;
  float: left;
  margin: 0 2% 0 0;
  @include layout-small {
    width: 100%;
    float: none;
    margin: 0;
  }
}
